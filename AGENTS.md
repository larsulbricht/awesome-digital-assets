# AGENTS.md

## Repository purpose
This repository is a curated awesome list for digital assets in financial services, with a strong European and institutional perspective.

## Editing guidelines
- Preserve the existing tone and structure of the README.
- Prefer institutional and ecosystem-relevant sources.
- Prioritize Europe, but include globally relevant sources when they materially affect European markets.
- Avoid low-substance vendor marketing pages unless they are strategically important market infrastructure announcements.
- Do not remove entries unless they are dead links, duplicates, not relevant enough or clearly outdated.
- Keep descriptions concise and neutral.
- Do not invent facts, dates, or summaries.
- If unsure, add entries to `candidates.md` instead of editing the main README aggressively.

## Review guidelines
- Flag dead links.
- Flag duplicate coverage.
- Flag categories with weak or outdated coverage.
- Suggest improvements to wording consistency.
- Prefer draft pull requests.

## Output format
When proposing changes:
1. Add a short change summary.
2. Separate into:
   - Add
   - Review
   - Remove/Fix
3. Explain why each proposed source is relevant.
