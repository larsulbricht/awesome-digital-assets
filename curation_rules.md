# Curation Rules

## Include
- Regulatory papers
- Consultation papers
- Important infrastructure announcements
- Institutional ecosystem maps
- High-quality explainers with lasting value
- Research reports
- Courses
- Tracker (e.g. News, Licences)

## Exclude
- Thin marketing landing pages
- Generic blockchain explainers without capital-markets relevance
- Duplicate links covering the same item
- News with no durable informational value

## Preferred metadata
Each item should ideally have:
- title
- link
- one-line description
- category
- rationale for inclusion
- date (if relevant)

## Decision logic
- If high confidence: propose README edit
- If medium confidence: add to candidates.md
- If low confidence: mention in PR notes only
