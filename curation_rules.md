# Curation Rules

## Include
Include content such as:
- regulatory papers
- consultation papers
- research reports
- institutional ecosystem maps
- important infrastructure announcements
- high-quality explainers with lasting value
- useful trackers, databases, or reference collections
- courses or educational resources with durable value

## Exclude
Exclude content such as:
- thin marketing landing pages
- generic blockchain explainers without capital-markets relevance
- duplicate links covering the same item
- short-lived news with no durable informational value
- low-substance opinion pieces
- broken or inaccessible links

## Preferred metadata
Each item should ideally have:
- title
- link
- one-line description
- category
- rationale for inclusion
- date, if relevant

## Decision logic
- High confidence:
  - propose a direct README edit
- Medium confidence:
  - add to `candidates.md`
- Low confidence:
  - mention only in PR notes

## Quality threshold
A source is a strong fit if it is at least one of the following:
- institutionally relevant
- analytically strong
- strategically important for market structure
- useful as a durable reference for practitioners

## Editorial principle
This is a curated list, not an exhaustive database.
Favor signal over volume.
